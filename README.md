# Geometry Dash Clone

A 2D platformer game inspired by Geometry Dash, built in Unity. Navigate through three challenging levels with different mechanics including gravity manipulation and ship transformation.

Link: (https://play.unity.com/api/v1/games/game/8965719a-8f5e-415c-ad5e-baa056de03b4/build/latest/frame)

### Installation
1. Download the game files from the releases page
2. Extract the ZIP file to your desired location
3. Select your player color from the main menu
4. Choose a level and start playing!

### Controls
- **Spacebar** - Jump (Cube mode) / Fly (Ship mode)
- **Mouse** - Navigate menus

### Game Modes
- **Cube Mode**: Automatically runs forward, press space to jump
- **Ship Mode**: Hold space to fly up, release to fly down

## ‚ú® Features

- **Three Unique Levels**
  - Level 1: The Beginning - Easy introduction level
  - Level 2: Gravity Shift - Medium difficulty with gravity portals
  - Level 3: Sky High - Hard difficulty with ship mode and mixed mechanics

- **Game Mechanics**
  - Gravity portals that flip gravity direction
  - Ship transformation portals
  - Inverted gravity works in both cube and ship modes
  - One-hit death system with instant respawn
  - Progress tracking

- **Player Customization**
  - 8 color options for player character
  - Color applies to both cube and ship forms

- **UI Features**
  - Main menu with level selection
  - Pause menu (Resume/Quit options)
  - Level completion screen
  - Dynamic progress bar
